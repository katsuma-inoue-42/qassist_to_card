# QAssist subprint to card

QAssistのサブプリントを分割してデッキ化するソフトウェアです。
Ankiでの使用を想定し、デッキをロードするためのCSVファイル (表ファイル)も同時に生成します。
家庭内で依頼されて作成したものを、個人学習の効率化の目的の下MITライセンスで一般公開しています。
